# Stock App

A responsively designed React project where brands and models can be filtered and sales and purchase rates are charted.

<hr />

## What I Used

- React Router Dom
- Material UI
- React Toastify
- Formik-Yup
- React Redux
- Tremor/react

### Installation Options

```
$ git clone https://github.com/HilalVildan/Stock-App.git
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/HilalVildan/Stock-App.git
```

```
$ yarn
```

```
$ yarn start
```
